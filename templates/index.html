About Drive Ezzy

Drive Ezzy is a modern cloud-based cab rental platform that allows users to browse vehicles, book rides, extend rentals, and manage reservations through an intuitive web interface.

AWS Cloud Services

The system is built using core AWS services to ensure scalability and reliability:

  • Amazon EC2: Hosting the Flask web application.
  • Amazon DynamoDB: Storing users, vehicles, and bookings.
  • Amazon SNS: Sending real-time booking notifications.
  • AWS IAM: Managing secure access and permissions.

Seamless Ride Experience

Users can easily filter vehicles (Sedan, SUV, Mini), modify booking durations, cancel or upgrade rides, and track all reservations from their personal dashboard.
