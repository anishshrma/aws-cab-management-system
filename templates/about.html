About Drive Ezzy

Welcome to Drive Ezzy, an AWS-Powered Cab Management System designed to simplify vehicle rentals and deliver a seamless ride-booking experience using modern cloud technologies.

Our Mission

Our mission is to provide a scalable, secure, and user-friendly cab rental platform that enables customers to easily browse vehicles, manage bookings, and modify reservations, while allowing administrators to efficiently manage fleet operations using AWS services.

Technology Stack

  • Frontend: HTML, CSS, and JavaScript for responsive and intuitive user interfaces.
  • Backend: Python Flask for handling authentication, bookings, and business logic.
  • Cloud Platform: Amazon Web Services (AWS) for scalable and reliable infrastructure.

Why AWS for Drive Ezzy?

AWS provides the reliability, scalability, and security required for a modern cab management system. Drive Ezzy leverages:

  • Amazon EC2: To host the Flask application with high availability.
  • Amazon DynamoDB: For fast, secure, and scalable NoSQL data storage of users, vehicles, and bookings.
  • Amazon SNS: To send real-time booking and activity notifications.
  • AWS IAM: For secure access control and role-based permissions.
